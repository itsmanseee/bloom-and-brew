# 🌿 Bloom & Brew — Botanical Café

A beautiful, fully responsive café landing page built as a Fiverr portfolio piece.

## Features
- Responsive design (mobile, tablet, desktop)
- Watercolour botanical SVG illustrations
- Real food photography (Unsplash)
- Animated hero with floating illustrations
- Interactive menu with category filtering
- Scroll-reveal animations
- Mobile hamburger navigation
- GitHub Pages hosted
